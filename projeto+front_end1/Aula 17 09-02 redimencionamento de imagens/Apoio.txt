Para o redimencionamento de imagens, utilizados a tag <picture></picture>, para colocar uma imagens precisamos por 
    ex: <picture>
            <img src="x"/>
        </picture>
Quando queremos então, redimencionar a imagem e mudar a escala, utilizamos a tag <source media="x"/>
    ex: <picture>
            <source media="(min-width:600px)"/>
            <img src="x"/>
        </picture>
Quando queremos então, que após o tamanho minimo, a imagem mude, pra outra, dentro da tag source, colocamos o minimo e srcset="x"
    ex: <picture>
            <source media="(min-width:600px)" srcset="y"/>
            <img src="x"/>
        </picture>

Para controle do background, podemos fazer no CSS pelo tamanho max ou min da tela 
    ex: @media (max-width: 700px) {
        background-color: #fff
    }
    @media (max-width: 100px) {
        background-color: #f00
    }
na tag @media, dentro CSS, podemos colocar alguns atributos, como:
    ex: @media only print{                              (usamos isso para escolher o que vai aparecer só na hora de imprimir)
        h1
    }
    ex: @media only screen{                             (usamos para escolher o que vai aparecer na tela quando abrirmos)
        h1
    }
    ex: body{
        display: flex
        } 
    @media (orientation: landscape){                    (faz com que, enquanto a pagina estiver "deitada" tudo apareça na horizontal) 
        body{
        display: flex
        } 
    } 
    @media (orientation: portrait){                    (faz com que, enquanto a pagina estiver "em pé" tudo apareça na vertical) 
        body{
        display: row
        } 
    }
    @media (min-aspect-ration:1/1) {                       (faz com que a tela siga o tamanho da proporção escolhida)
        body{
            background-color: #ff0
        }
    }
