Flex-box
    Alinhamento de divs
    -precisamos de uma div mae, normalmente chamada como <div class="container">
    - no CSS, precisamos chamar a div mae <div class="container"> e colocar o atributo
    .container{
        display: flex;
    }
    Podemos mudar a direção da flexbox usando outro atributo
     .container{
        display: flex;
        flex-direction: row;    (linha padrão)
        flex-direction: row-reverse; (inverte a linha, começando da direita pra esquerda)
        flex-direction: column; (padrão da programação)
        flex-direction: column-reverse; (inverte a coluna)
     }
    Para que a linha se quebre e se divida, quebrando a pagina usamos o wrap
     .container{
        display: flex;
        flex-wrap: wrap;       (adapta a linha, dividindo conforme a pagina diminui)
     }
    Para escolher o tamanho da div filha dentro da div pai, podemos usar o atributo dentro da
     tag filha
     .teste1{
         flex:1; (usando o 1, ela vai ocupar o maximo da tela, se usar 2, 3, 4, 5, o tamanho
         se dividir permanencendo maior conforme o numero, igual %)
     }
    A tag que substitui os dois é a flex-flow
     .container{
         display: flex;
         flex-flow: row wrap;
     }
    Para realizar o alinhamento usamos a tag justify-content, align-items:
            justify-content por linha; align-items por colunas;
            *sempre cuidado com o flex-flow.*
     .container{
         display: flex;
         justify-content: center; (centraliza todo o conteúdo)
         justify-content: flex-start; (alinha começando pelo lado esquerdo)
         justify-content: flex-end; (alinha começando pelo lado direito, diferente do reverse)
         justify-content: space-around; (separa as divs, mantendo paralelas)
         justify-content: space-beteween; (espaço entre as divs, o mais longe possivel)
         align-items: center; (alinha a coluna no centro)
         align-items: flex-start; ()
         align-items: ; ()

     }
    Para alinhar apenas uma das divs do container, damos uma class e colocamos o flex-self
    Para escolher organizar, podemos colocar por order
        - order: 1;
        por default todos são grupo 0.
