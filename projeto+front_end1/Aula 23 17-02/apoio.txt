HTML - SEMANTICO
    - fazer uma lista pra assessibilidade
        <dl>
            <dt>Cafe</dt>
            <dd>Preto e quente </dd>
            <dt>Leite </dt>
            <dd>Branco e quente ou frio </dd>
        </dl>
        Sendo <dd> </dd> a tag de descrição do item da lista
        pode ser usada dentro no <dd></dd> a tag <ul><li></li></ul>
    - citação
        para apenas uma linha:
        tag <q></q> poe entre aspas para acessibilidade e busca facilita a busca
        para texto grande:
        tag <blockquote cite="www.pipipopo.com.br"> </blockquote>
    - main
        tag de conteúdo principal da pagina, igual a tag <article>
        mas só pode ser utilizada uma vez.
    - address
        tag que fica dentro do <footer></footer>
        a tag responsavel para ficar com os dados da pagina de contato da pagina.
    - figure
        tag para colocar uma imagem
        <figure></figure>
        podendo colocar a <figcaption>utilizada para descrever a imagem</figcaption>
