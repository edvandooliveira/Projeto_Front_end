3 formas de inserir o CSS no HTML

1- CSS Inline
2- CSS Interno
3- CSS Externo

1- CSS Inline
    - colocando os atributos dentro das tags, como por exemplo:
        style="color: rgb(6, 34, 6); font-size: 19px"
2- CSS Interno
    - Colocando no header o comando <style> e adicionando quais tags eu quero colorir adicionar 
    atributos.
        <style> h1 {color: olive; font-style: oblique; font-size: 28px;}
            h2 {font-size: 25px; color: black; font-weight: bold; }
            h3 {font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; color: lightgreen}
        </style>
3- CSS Externo
    - Arquivo criado fora do html, um segundo arquivo, podendo ter qualquer nome. Mas usualmente é 
    chamado de style.CSS
    - no header colocar o comando 
            <link rel="stylesheet" href="style.css">
    - sendo ele, <link> o que liga um arquivo no outro, rel="" o tipo de relação entre os arquivos,
    e href="" o nome do arquivo/endereço do arquivo.
    - dentro do arquivo .css, colocar as tags utilizadas, e estilizar elas dentro do arquivo, esco-
    lhendo cor, tamanho da fonte e afins.
    - para especificar, por exemplo dois <h1> colocar o atributo dentro, sendo ele o id="" e colocar
    o nome que preferir.
    - dentro do arquivo .css, colocar # e o titulo que você colocou no id=""
        Exemplo: dentro do html
                <h1 id="titulo"> pipipopo </h1>
                 dentro do arquivo .css
                 #titulo {}
    - utilizar tambem no lugar do id="", utilizar o class=""
        Exemplo: dentro do html
                <h1 class="titulo2"> pipipopo </h1>
                 dentro do arquivo .css
                 .titulo2 {}
    - Pode-se colocar os mesmos atributos em varias tags.
        Exemplo: dentro do html
                <h1 class="titulo2"> pipipopo </h1>
                <h1 id="titulo"> pipipopo </h1>
                <h2> aaaaa </h2>
                 dentro do arquivo .css
                 .titulo2, h2, titulo {}
    -   para trocar cores de listas, como exemplo o comando <ol> e o <li>, devesse colocar no .css
        Exemplo: ol il {}
