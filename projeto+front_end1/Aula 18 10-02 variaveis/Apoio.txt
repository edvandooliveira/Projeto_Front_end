As variaveis de adaptação do site  
    - aplicamos no CSS, dar prioridade para colocar a variavel dentro da maior tag possivel, no html, ou body
        como para trocar a cor, fonte ou qualquer outra coisa de varios itens podemos criar uma variavel e adicionar uma cor a ela
        ex: :root{                                                  para criar uma varivel, sempre temos q colocar -- e o nome que
            --bg-color: #ff0                                        quiser.
            --font-color: #f00
            --font-size: 25px
        }
        para aplicar a variavel nas tags, colocamos no lugar da cor a variavel, sempre precisamos por o var antes da variavel.
        ex: h1{
            background-color: var(--bg-color)
            color: var(--font-color)
            font-size:30px
        }
        podemos usar as variaveis para diminuir ou alterar algo dentro do @media
        ex: @media (max-width: 260px) {
                :root{                                                  
                --bg-color: #ff0                                        
                --font-color: #f00
                --font-size: 15px
                }
            } 
        Para adicionar um video do youtube para o site, para ele ficar responsivo precisamos dentro do html adicionar
        ex: <div class="video">
                <div class="video-area">
                    <iframe x>
                </div>
            </div>
        E depois dentro CSS, precisamos alterar as variaveis
        ex:
            .video{
                width: 100%;
                max-width: 450px;
                margin: auto;
            }

            .video-area{
                position: relative;
                padding: 0px 0px 56.25%;
            }

            .video-area iframe{
                position: absolute;
                top: 0px;
                bottom: 0px;
                left: 0px;
                width: 100%;
                height: 100%;
                border: 0px;  
            } 
            Para fazermos a relação de adaptatividade com imagens
            ex: img{
                width: max(50% , 300px); 
                width: min(50% , 300px);
                width: 150px;
            }
            Para utilizar a função de calculo, chamamos a class no CSS, sempre mantem o valor de pixels definidos fixos, menos a
                porcentagem escolhida.
            ex:.container{
                width: calc(100% - 50px);
                height: 200px;
                background-color: aquamarine;
            }
