-Formatação do formulario com CSS
    - O label por padrão , já segue tudo na msm linha, colocando block ele pula uma linha.
        label{
            display: block
        }
    - para a customização das areas no CSS de diferentes inputs, podemos colocar
        input [type: "number"], [type: "name"], textarea, select.

- Validação nativa do HTML
    - campos obrigatorios:
        dentro do input, colocamos o atributo "required"
        ex: <p>E-mail para contato</p>
            <input type="email" name="email" required/>
            <p>Data de inicio de sintomas</p>
            <input type="date" name="data de inicio" required/>
            <p>Data da avaliação</p>
            <input type="date" name="data de avaliação" required />
    - quantidade minima de caracteres
        dentro do input, colocamos o atributo "minlength"
            <input type="text" name="nome" minlength="5"/>
    - quantidade maxima de caracteres  
        dentro do input, colocamos o atributo "maxlength"
            <input type="text" name="nome maxlength="75"
    - para o campo number, usamos só max=""
             <input type="number" name="nome max="75" />
    - para o campo date, para colocar maximo e minimo de idade
             <input type="date" name="data" min="1900-01-01" max="2004-12-31"/>
