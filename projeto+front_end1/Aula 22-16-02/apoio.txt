CSS 3
    Para centralizar, usar o margin: "auto"
    -Degradê
        -background: linear-gradient(x, y, z)
        x: cor inicial
        y: cor final
        z: angulo pra onde a cor vai fazer o degrade, seguindo angulos
            90deg, -90deg, 45deg, -45deg, escolhendo pra onde ficar a cor.
    -animação
        -@keyframes identifier
            (identifier pode ser escolhido com nome que quiser)
            0% (background-color: x, top: 0, left:0)
            25% (background-color: y, top: 0, left:200px)
            50% (background-color: z, top: 200px, left:200px)
            75% (background-color: a, top: 200px, left:0)
            100% (background-color: b, top: 0, left:0)
            (escolhemos por frames como deve ocorrer a animação,na selecionada,
             as cores são para mudar conforme a animação corre)
            (top, left, usamos o posicionamento para mover a animação.)
        -Depois dos frames marcados, vamos na tag que é para se animada e chamamos
        a animação.
            div{
                animation-name: teste; (nome da animação)
                position: absolut
                animation-duration: 4s; (quanto tempo demora a animação)
                animation-delay: 3s; (após entrar no site para terminar de carregar)
                animation-iteration-count: infinite; (loop na animação infinita)
            }
    -sombreamento
        - chamamos o atributo box-shadow.
            box-shadow: a b c d
            a: sombreamento direita-esquerda.
            b: sombreamento cima-baixo.
            c: densidade do sombreamento.
            d: cor do sombreamento. 
        - para sombreamento do texto
           text-shadow: a b c d
            a: sombreamento direita-esquerda.
            b: sombreamento cima-baixo.
            c: densidade do sombreamento.
            d: cor do sombreamento.
    -transição
        - mudar algo de lugar, ou cores, ou posicionamento
            transition: a b c
            a: all
            b: tempo
            c: linear 
        -podemos usar com o hover
    -quebratexto (word-wrap)
        -word-wrap: none; (nao muda nada)
        -word-wrap: break-word (quebra texto)

Html (tag meta)
    - para fazer aquela imagem que aparece quando vai enviar link
    - no <head> usamos  a tag meta, sendo ela
        <meta property="og: url" content="www.sistemasenai.com.br/teste123"/>
        <meta property="og: type" content="article"/>
            article= noticia
            website= site
            product= produto
            video= video.
        <meta property= "title" content= "titulo do site"/>
        <meta property= "description" content= "subtitulo e texto do link"/>
        <meta property= "image" content= "link da fotinha que vai com link"/>
